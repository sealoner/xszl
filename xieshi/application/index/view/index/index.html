关于谢氏制冷

谢氏制冷 —— 专业的冷库设备生产制造商

谢氏制冷科技有限公司是一家专门生产销售冷库设备，冷库板，冷库门，全套冷库设备，小型冷库设备，冷风机，保鲜冷库的专业冷库设备厂家，多年来公司生产的玻璃钢彩钢板，不锈钢系列装配式冷库广泛用于各大超市，宾馆以及海鲜水产保温场所，并以产品质量，贴心的售后服务，合理的市场价格，良好的公司信誉赢得广大客户的认可。

主要产品有：制冷设备，冷库保温板，聚氨脂板，中高低温冷库，低温实验库，速冻库，速冻隧道，单螺旋和双螺旋速冻机，真空预冷机，制冰机等
